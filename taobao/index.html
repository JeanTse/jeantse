和同事合租时对方不知不觉吃了我很多安利，而且都是从此往后再也离不开的日常小玩意儿。
反正感觉自己在淘货方面真的有点东西。不归纳出来浪费了！

日用品

收纳

  • 天天特卖工厂店的抽绳束口袋，各种尺码。
    啥都能装，便携，一定程度防水。
    万能使用！去健身房装鞋袜衣服矿泉水，去游泳馆做干湿分离，出差旅行分类压缩，甚至日常买菜取快递！

  • tb搜管理药物专业经营店，专卖各种密封药盒。
    除了装药还可以装硬糖装蛋白粉，放几个月都不受潮。
    推荐10ml的迷你款，虽然外观朴实，但比那些花里胡哨的塑料小盒靠谱多了！

  • 论斤卖的吸水抹布！是那种很高档的可以用来擦车擦镜片拖地洗碗甚至当擦手巾都OK的纤维吸尘布。
    有各种尺寸各种纤维长短的型号，最大的尺寸有A2，最小的尺寸差不多A4。全都有缩边，完全不会掉屑的。质地和进口超市的高档纤维抹布完全一致（本来就是出口货），价格相差数量级。
    反正我当年脑子一热买了5kg，分给同事一半，剩下的还用了好几年至今没用完……我买的是小熊一家旗舰店的。

  • 太力真空收纳袋，最近更新的萌熊款是真的颠覆了以前旧款的使用感受。强烈推荐！和旧款完全不是同一种东西了的感觉。

其他

  • 3M超能胶，0.5g装。
    什么都能粘，而且粘得无比牢。比如我电动车的电瓶塑料外壳摔出的裂缝、鞋底、手机便携指环的背胶加固……等等等等。
    关键最好的是它用过一次不会凝固！0.5克可以用很久很久，每次都能顺利挤出来。缺陷是挤出来之后涂上需要等好几分钟才会逐渐凝固，然后等半小时以上才能固化完全。所以要有耐心。

  • 得力小学生用坐姿架（就是托下巴的架子）。
    公司午睡不需要再弯腰趴着压到手麻。在架子上垫一层抱枕，然后下巴和胸口都能压着松软的抱枕双手环抱无压力入眠了！睡完从桌沿取下来往抽屉里一丢谁都发现不了。

  • 箱装的克林海绵，100p约30元。买2-3cm厚度的最好用。

  • 蓝月亮蓝泡泡。放马桶水箱的那种。
    贵的便宜的我都尝试过很多厂家很多品牌，结果还是蓝月亮的靠谱。虽然价格贵，但一块真的能用很久很久……久到我都会忘记它的存在，猛然有天突然发现冲水不蓝了才放新的进去。而且确实能起到不小的清洁效果。虽然长期还是需要额外清洗，但日场累积污垢的速度真的变慢很多。

  • 天天特卖工厂店的强力透明挂钩，批发。或者去1688不但可以买到普通款，还有各种异形款。
    偶尔我们买的挂墙商品自带的凹槽之类的损坏了，就可以从1688找到配套的异形款，非常便宜。

服饰鞋包

衣裤

  • 内裤=tb搜木子棉店。闭眼挑也不后悔。
    每人喜欢的材质款式不同没法具体推荐款型，但所有款型我几乎都买过，都靠谱！而且同款一定是tb价格最低的（如果有价格更低的貌似同款，基本质量都有差）

  • 袜子专卖店请搜newsocks。
    闭眼买！任何品种都特别好穿！性能极佳！价格白菜！私人推荐各种长筒袜和运动健身袜。

  • 运动内衣，搜奔奔兔 外贸铺。
    杂款清仓的9.9元白菜价运动内衣我已经陆续囤了5件，就日常穿都非常舒适~
    其他衣裤有合适尺码的话也值得入，质量都很靠谱。

  • 6IXTY8IGHT的前扣系列，每年促销都闭眼囤！不过它家更适合小胸。

  • tb搜美斯特邦威官方outlets店。
    常年二三折，而且近年来他们的衣服一点都不土了！！只要能找到合适尺寸简直能闭眼入。

鞋包

  • 热缩膜鞋子保护袋。
    吹风机一吹就把鞋子搞成类似真空包装了！真的很省心。

  • 慧慧家外贸大码鞋、搜鞋屋大码鞋和毛豆大码鞋。
    3店虽然风格不同，性价比却不相伯仲。关键鞋子质量都很顶！

  • 标奇擦鞋湿巾。比其他品牌的好用！可以放心囤。

  • 猫咪布包：ESQ和KineCat。
    自从曾经单肩皮包背成肩周炎后，我就彻底投入轻巧布包的怀抱。最近10年入手的都是这两家品牌的各种尺寸各种造型的包包、挂饰和抱枕。反正就是都太可爱了！！！

  • 各种内胆包，随便搜。有软质的，有毡布硬挺的，按需选择，注意尺寸。
    但凡单价超过20元都是宰客，找同款是个好功能。

床上用品

  • tb门氏家纺工厂店的长条冰丝凉感抱枕~
    夏日催眠极品！也有暖绒款等不同材质，适合孕妇。

  • 打针的那种被子固定器。
    虽然拉扯下难免会留下针孔，但效果好啊！
    而且用途远不止固定被子。我最常用是固定不同尺寸的枕头和枕套。还可以订窗帘多余的长度，学待售品把袜子内衣等钉起来收纳，做针线时先钉出轮廓再慢慢缝避免移位，调整电动车头盔的系带长度等等。

  • 超冷感薄被，tb搜阳光外贸批发。质量靠谱价格实惠。推荐双面款！

饰品

  • 1688的义乌市宝钰饰品有限公司。
    反正tb热卖款都不超过10元，长草就全买下！每次批发一大包和亲友分享。戴几次就闲置，脏了磨损了氧化了都不心疼。

  • 红木吧的瑕疵随机混搭雕花檀木梳，反正我没看出哪里有瑕疵！20元5把，各种大小各种粗细的搭配，我所有场合的梳子都解决了。
    甚至订单备注自己想要的长短和粗细，都会按照你的要求搭配，并不完全随机。

  • 花式发簪我只买竹影林曦。因为真的便宜又好看！完全没有小圈子溢价。

  • 18cm左右长度的基础款直发簪，建议tb搜儿童训练实木短筷。
    同样材质的性价比只有发簪店同款的几分之一！推荐红檀木，最好选尖头，或买到手后用磨刀石磨尖。

饮食

吃的

  • 蛋白粉，tb搜康莱食化商贸店，浓缩乳清蛋白原料，含量高达80-90%。
    性价比远高于其他品牌。健身增肌或给老人补充蛋白质都OK。我喜欢原味的，加入麦片等食物之中不会影响口味。

厨具

  • 铸铁平底荷包蛋小油锅。
    10cm款刚好煎一个蛋，是真的完全不粘锅！超完美又便宜~远胜各种家居品牌的煎蛋平底锅。

  • 麦饭石炒锅yyds。不粘锅，而且轻巧能够单手颠锅。

  • 竹子吸管。长短和口径任选。
    安全卫生环保耐热。清洗的话建议到1688批发一组试管刷…当然我是直接丢洗碗机。

  • tb豪然旗舰店的碗碟套装。
    风格简洁，美观大方，价格只有其他店的几分之一。发货包装也很严实，开箱碎裂无条件补发。我是选了一套纯白黑边的超有调调。

美妆

化妆

  • 亨丝1818拉线眉笔，这是唯一能令我放弃资深堂六角眉笔的选择。
    虽然需要用附带的刀片削笔芯（我是copy砍刀眉笔的形状，而且削得非常尖细可以画出一根根眉毛），但因为笔杆是拉丝剥离的纸片所以不用削木头，然后笔芯又是软质，就非常好削，也非常好上色。
    关键是便宜啊！哪怕为了保持尖细每天削掉很多，20元买一盒能用一整年……
    缺点当然有，那就是容易上色也意味着容易被蹭掉，所以建议用眉毛雨衣或其他定妆产品。

  • 既然已经提到了眉毛雨衣，那么大创DAISO眉毛雨衣我只推荐最早的经典款。
    后续新款效果都不好，不是拔干紧绷就是反光。但就实体店都断货，代购也很多假货，所以推荐大创的日本官网用跨国转运，官网是量贩的10瓶装，加运费平摊下来单价不比代购贵。

  • tb俄罗斯田园猫的指甲油，同样是闭眼入……
    各种磁吸猫眼变色款和炸裂款都美爆！然后速干顶油也很好用。它家基本是同款淘宝最低价了。

  • 卸甲油我选ZOYA的960ml超大瓶，搭配丽丽贝尔的大款卸妆棉。便宜效果好。

  • bifesta或曼丹的眼唇卸妆液+500ml的日本PUREVIVI皇后卸妆水+丽丽贝尔大款卸妆棉。
    无敌高效又便宜的敏感肌卸妆套餐！！

  • 歌剧魅影单色遮瑕膏，绿色遮红血丝。
    用指甲盖挑一点掺入自己习惯的粉底液，效果完美自然又持久！
    4g一小罐，每天上班都用的话也不至于吃灰变质。连续空罐中。

  • 真空粉底液分装瓶。不光适合装粉底液，用酒精消毒之后还能装川贝枇杷膏、蜂蜜之类的膏状食物。总之是便携神器！

护肤

  • 美多丝护发精油，比欧*雅以及某韩国爆款精油好用多了！
    吸收快，不沾手。完全没有假滑的油腻感！我抹完头发都不用洗手。不像其他品牌精油不用洗手液都洗不干净。
    从来都节省不肯多买东西的父母回老家还特意找我代购~真的是被营销耽误的真材实料的优秀产品！

  • Oralabs果味迷你润唇膏，只能从香港代购。
    保湿性好不油腻，果香宜人（曾在公司被老鼠叼走过不止一支），关键是便宜又便携，只有指节大小。每年双11我都要囤一大盒。
    但疫情之后就没处买了……现在只能换用贝德玛润唇膏，3支装打折差不多40元，同样是轻薄不油腻的风格。比超市开架的某些知名品牌舒适多了。

  • 荘士蛇油膏。
    20年前在大学的教育超市偶然买了一次就念念不忘，但毕业后实体超市再也找不到，后来tb兴起但也没有卖家。这次双十一凑单无意中搜到居然有代购！到货后确认就是当年的味儿~
    它的流变力学性质真的很独特，刚擦觉得很厚重很腻，但只要多搓揉几秒就会突然液化然后迅速被皮肤吸收，秒变干爽！随后洗手沾水都不会有那种手上一层油脂被水冲下来的滑腻感，洗手时就像没擦过护手霜一样清爽。
    而且保湿效果极其持久！不洗手的话多久都不会被蹭掉变干燥。
    唯一的缺点是“香”味感人……但搓揉吸收之后就闻不到味道了。我现在甚至直接用它当面霜……反正就是无比神奇的万能保湿乳。

  • 丝塔芙敏感肌洗面奶=Gentle Skin Cleanser。
    温和不起泡，是要无水时直接搓脸然后用水洗净。适合早晨无需深度洁面时使用。
    理论上也可用作卸妆乳，乳化擦除简单底妆没问题。
    500g×2瓶的超大组合可以用1整年也才150元不到。

  • HABA烷鲨精油+G-lotion=中老年干皮福音

洗浴

  • 擦手纸，就是商场WC那种大张扯下来擦手的厚实吸水纸。
    家里用也很方便，特别是擦脸不再需要毛巾，避免潮湿生菌，而且不掉屑，比餐巾纸更适合大面积擦干。
    推荐tb搜云端生活家居专营店的清风擦手纸10包套装。其他网上超市都只有不知名品牌而且都很贵。我用了好几年，现在只有这家卖清风的卖得最实惠。

  • 在1688搜长条尼龙搓澡巾，我推荐南通臻达特种纤维有限公司。
    谁用谁知道……太爽快了！沾水之后搓揉会变柔软，起泡丰富，搓后背去痘去死皮太爽了。
    然后又很便宜，出门旅行当作一次性用品，或者包裹香皂去游泳健身也都很方便。还可剪成小块洗脸洗手。

  • tb的天真小白鲜果食物洗发皂。
    它家的羊奶洁面沐浴皂我是无限回购！洗澡完全不紧绷，同时也能洗脸卸妆，比用过的无数大牌小牌的洗面奶都更舒服。搭配赠送的起泡网以及上面推荐的长条尼龙搓澡巾，真的无比惬意。
    至于洗发皂，每个人发质不同，所以即便我推荐其中的椰子皂，也不见得适合你。还需每种都买来亲自尝试。

  • 乐德牙线。
    特别细，特别韧。超越一切国内外知名品牌。大量囤货！据说他就是国外某高价牙线的OEM生产厂家。

  • 便携折叠按摩拖鞋，去游泳馆真的轻便不占空间。

隐形眼镜

  • 品牌=SEED实瞳；日产型号=京樱View support；台产平替=幻樱S。
    没啥好说的。我花了几千元把国内外各知名品牌的主要型号全都试过一遍，只有SEED完全没有任何缺点。就连价格都不贵！

  • 便携盒=eyekan旗舰店。
    其中可爱牛动物系列刚好可装SEED日抛2对（4p）+夹子+小条眼药水，空间利用率最高。
    如果用月抛则优选姜饼人的双盒系列容量大，可装1个护理盒+2瓶小护理液+夹子+小条眼药水，或2个护理盒+1瓶小护理液+夹子+小条眼药水。

电器

小家电

  • USB充电的人体感应式小夜灯推荐两款：
    公牛是磁吸式，优点是容易摘取，缺点是发光直接照射前方有点刺眼，而且感应范围大约只有左右45度角。但价格最便宜。
    美的是插片式，摘取较麻烦。但它是后侧发光，光线不直射眼睛，所以非常柔和，照射范围也更广，感应范围更是超级无敌宽！虽然贵了一丢丢但物有所值。价格也就40多。

  • 海尔智能马桶V-5225。
    性价比无敌高，功能齐全，包括遥控器+操作旋钮，而且也是即热型。形状也更适合中国人体型。和任意款马桶的适配度也特别高，基本通用尺寸的都能装。
    我自己原本是在多次的日本旅行中更偏好toto，但结果用下来发现海尔这款真的比toto各种型号的都更适合自家用！国产家电yyds！

  • 小熊的mini电饭煲，一人食王道。

  • 小熊的超声波清晰机。
    和其他品牌的超声清洗相比，它振动非常细密，肉眼甚至看不到水波晃动，只能看到有气泡附着在脏处不断冒出来。洗完就很干净，运行时的噪音甚至不比蚊子嗡嗡叫更大！
    比那些乍一看工作十分辛苦动静特别大，但完事感觉不出啥效果的品牌靠谱多了。
    新款还增加了专洗隐形眼镜的小仓。

手机配套品

  • 柏奈尔手机贴膜，送贴膜神器。
    我的手机这么多年升级了好几次但每次新型号也都在它家继续回购，于是神器都囤了一抽屉…是真的好贴好用又便宜！

  • 千致手机壳，无边框硅胶款。
    主要是不会覆盖侧边按钮，这样操作按钮的手感最棒。四角包裹的保护性也极好，经常摔也很少摔坏钢化膜，都是边框四角自己先开裂。

  • 易亮镜片清洁纸。用过的擦镜湿巾中效果最好的。

  • 热敏迷你打印机（我买的喵喵机）。
    这玩意儿说有用吧用的次数很少；说没用吧日常打取件码打各种型号的标签贴甚至打菜谱打小地图…都特么确实很实用！

虚拟产品和app

  • 天猫88VIP，1年88元=优酷+网易云+饿了么+高德半年VIP+天猫自营95折。感觉是在做慈善= =||||

  • 时光序app。
    整合了日程提醒、习惯打卡、生理期、体重和饮食纪录、倒数纪念日、番茄时钟、课程表、排班表、简单记账、作息管理……等等无数功能。
    关键是自定义程度特别高！然后一次性VIP又不贵。直接替代了我以前用习惯了n个各类app。

  • 白描OCR文字识别app。
    免费版就足够日常使用了！工作大批量使用买终身VIP也不贵。

  • longshot自动拼长图app。
    虽然华为系统自带截长图功能，但很多情况却还不支持。这个拼图可以自选多图拼接，拼接位置也可以自己调整。反正就是很方便。

  • See微博。
    虽然现在被官方禁止提供app市场下载，只能私下apk安装，但真的超级好用！

  • Alreader + 讯飞语记的TTS设置为男主播。
    这个搭配是txt文本的小说语音朗读绝配！完败各书城自带的朗读。

  • vpnsoso.com + BifrostV（安卓） + V2rayN（Windows）=全平台科学上网。
